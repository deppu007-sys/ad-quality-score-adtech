# 📊 Ad Quality Score (AdTech Project)

## 🚀 Overview
This project calculates Ad Quality Score based on:
- CTR (Click Through Rate)
- Ad Relevance
- Landing Page Experience

## 🧠 Formula
Quality Score = (CTR Score + Relevance Score + Landing Score) / 3

## 📊 Features
- Calculates CTR
- Normalizes score to 10 scale
- Ranks ads based on performance

## 🛠 Tech Used
- Python
- Pandas

## 📈 Output
Ranks ads from best to worst based on Quality Score
